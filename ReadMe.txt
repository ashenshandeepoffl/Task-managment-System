# Task Management System

This is a simple Task Management System web application that allows users to manage tasks. 
It includes features such as adding tasks, marking tasks as completed, clearing all tasks, and displaying a live calendar with the current time.

## Getting Started

To run this project locally, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/ashenshandeepoffl/task-management-system.git
   ```

2. Navigate to the project directory:

   ```bash
   cd task-management-system
   ```

3. Open the `index.html` file in your web browser.

## Usage

- Add a new task: Enter a task in the input field and click the "Add Task" button.
- Mark a task as completed: Click the checkbox next to the task.
- Clear all tasks: Click the "Clear All Tasks" button.
- Live Calendar and Time: The current date and time are displayed on the page.
